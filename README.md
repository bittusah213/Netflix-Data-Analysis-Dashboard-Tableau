# Netflix-Data-Analysis-Dashboard-Tableau

## Project Overview

This project is an interactive Netflix Data Analysis Dashboard built using Tableau. The dashboard provides insights into Netflix's catalog of Movies and TV Shows by analyzing content type, ratings, genres, release years, countries, and yearly distribution. It enables users to explore Netflix's content library through interactive filters and visualizations.

## Objective

#### The goal of this project is to analyze Netflix's content and answer business questions such as:

#### What type of content dominates Netflix?
#### Which countries produce the most Netflix content?
#### What are the most common content ratings?
#### Which genres are most popular?
#### How has Netflix's content library grown over the years?
#### What are the details of a selected movie or TV show?
